# 🎃 HALLOWEEN 2026

HALLOWEEN 2026 is an interactive Halloween event platform created to manage a real-life party experience with teams, games, live scoreboards, costume voting, and admin controls.

The project was built with PHP, MySQL, JavaScript, HTML, and CSS, featuring a fully responsive UI inspired by modern Dribbble-style interfaces.

## ✨ Features

- 🔐 User authentication system
- 👑 Admin panel
- 🎲 Random team generator
- 🧠 Team leader/fixed player system
- 🏆 Real-time live scoreboard (without page reload)
- 🎮 Dynamic games page with modal details
- 🎭 Costume photo upload and voting system
- 📊 Individual and team scoring
- 📱 Fully responsive design
- ⚡ Real-time updates using AJAX
- 🎨 Custom Halloween-themed UI/UX

## 🚀 Purpose

The platform was developed to transform a traditional Halloween party into a competitive and interactive experience, combining games, drinking challenges, team rankings, and social interaction in a fun and immersive way.
